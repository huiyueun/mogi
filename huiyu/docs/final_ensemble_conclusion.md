# 최종 앙상블 결론: 왜 1등과 2등을 더했는가

작성일: 2026-07-01

## 결론 요약

이번 실험의 최종 결론은 다음이다.

> 1등 GOH30은 이미 강한 내부 앙상블이기 때문에, 같은 방향의 후처리나 비슷한 전문가를 추가해도 점수가 잘 오르지 않았다.  
> 반면 2등 코드는 GOH30과 다른 baseline, 다른 물리 가정, 다른 보수적 보정 성향을 갖고 있었고, 특히 급회전/고노이즈/z축 변화 구간에서 GOH30이 보지 못한 방향의 예측을 제공했다.  
> 그래서 2등 코드를 외부 expert로 붙였을 때 기존 GOH30 내부 앙상블만으로는 얻기 어려운 다양성이 생겼고, 그 결과 리더보드 점수가 개선된 것으로 해석된다.

최종적으로 확인된 최고 조합은 다음이다.

```text
final = 0.80 * GOH30_ODE_heavy + 0.20 * second_final
```

여기서:

```text
GOH30_ODE_heavy = 0.20 * GRU + 0.60 * ODE + 0.20 * H
second_final    = 0.60 * second_base + 0.40 * second_phys
```

즉 최종 구조는 다음과 같이 볼 수 있다.

```text
최종 제출
= 1등 GOH30 내부 expert 앙상블
+ 2등 외부 expert 앙상블
```

## 제출 점수 흐름

| 제출 | Public | Private | 해석 |
| --- | ---: | ---: | --- |
| GOH30 원본 | 0.7020 | 0.7025 | 1등 원본 코드 재현 |
| GOH30 ODE-heavy | 0.7018 | 0.7033 | GOH30 내부에서 ODE 비중 강화 |
| ODE-heavy 80% + 2등 final 20% | 0.7034 | 0.7042 | 현재 확인된 최고 조합 |

중요한 점은 ODE-heavy만으로도 private 기준 개선이 있었지만, 가장 큰 개선은 2등 final을 20% 섞었을 때 나왔다는 것이다.

## 왜 단순 후처리는 잘 안 먹혔나

처음에는 여러 후보정과 학습 전 아이디어를 시도했다.

- high_speed Kalman 보정
- high_noise / hard_turn / recent_turn / high_acc / vertical_change regime 구분
- OOF-lite 기반 GRU/ODE/H 비율 탐색
- H-heavy 조합
- KMeans cluster feature injection
- cluster별 sample weighting
- TCN expert
- Transformer-lite expert
- learned gating / stacking

이 중 일부는 OOF-lite에서는 좋아 보였다. 예를 들어 H 모델, TCN, Transformer-lite는 로컬 검증에서는 기존보다 높은 hit을 보였다.

하지만 리더보드에서는 대부분 전이되지 않았다. 이유는 GOH30 원본 자체가 이미 다음을 포함한 강한 앙상블이었기 때문으로 보인다.

- GRU 10개
- Neural ODE 10개
- 물리 기반 H 모델 10개
- 등속 외삽 기반 residual 학습
- local frame 변환
- 속도/가속/저크/각속도 피처
- 모델 평균을 통한 분산 감소

즉 GOH30 내부에는 이미 “최근 움직임을 따라가는 전문가”, “동역학을 보는 전문가”, “물리 prior를 보는 전문가”가 들어 있었다. 그래서 같은 데이터를 비슷한 관점으로 다시 본 후보들은 새로운 정보를 충분히 제공하지 못했다.

## 1등 GOH30의 성향

분석상 GOH30은 최근 관측된 운동 변화에 민감하게 반응하는 쪽이다.

특히 다음 신호를 적극적으로 사용한다.

- 최근 속도
- 최근 가속
- 저크
- 각속도
- 직선성
- 노이즈
- local frame 기준 상대 이동

ODE-heavy는 이 GOH30 내부에서 ODE expert의 비중을 키운 형태다.

```text
원본 GOH30 = GRU 33.3% + ODE 33.3% + H 33.3%
ODE-heavy = GRU 20%   + ODE 60%   + H 20%
```

따라서 ODE-heavy는 완전히 새로운 모델이라기보다, GOH30이 원래 갖고 있던 “동역학 기반 예측 성향”을 더 강하게 만든 것이다.

이 방식은 private에서 어느 정도 개선됐지만, 여전히 GOH30 내부 시야 안에서의 조정이다.

## 2등 코드가 제공한 다른 시야

2등 코드는 최종적으로 다음 구조였다.

```text
second_final = 0.60 * second_base + 0.40 * second_phys
```

분석 결과, 2등은 GOH30과 다른 예측 성향을 보였다.

### 1. 급회전/고노이즈 구간에서 덜 꺾음

GOH30 원본 대비 2등 final은 다음 구간에서 평균 회전각을 줄였다.

| 구간 | 2등 final - GOH30 원본 |
| --- | ---: |
| 전체 | -0.573도 |
| hard_turn | -1.474도 |
| recent_turn | -1.419도 |
| high_acc | -1.201도 |
| high_noise | -1.159도 |

이 말은 2등이 GOH30처럼 최근 회전 신호를 그대로 강하게 따라가기보다, 일부 구간에서는 더 보수적으로 궤적을 부드럽게 잡았다는 뜻이다.

### 2. z축 이동을 약하게 안정화함

2등 final은 GOH30 원본보다 z 이동을 평균적으로 조금 더 작게 잡았다.

| 구성요소 | z 이동 변화 | CV z에 더 가까운 비율 |
| --- | ---: | ---: |
| `second_base` | -0.000060 | 0.552 |
| `second_phys` | -0.000033 | 0.490 |
| `second_final` | -0.000070 | 0.551 |

특히 z 안정화는 `second_phys`보다는 `second_base` 또는 `second_final`에서 더 일관적으로 나타났다.

### 3. ODE-heavy와 같은 방향 보정이 아님

ODE-heavy가 GOH30 원본에서 움직인 방향과 2등 구성요소가 움직인 방향을 비교하면, 평균 cosine이 음수였다.

| 구성요소 | ODE-heavy 보정과 평균 cos | 같은 방향 비율 |
| --- | ---: | ---: |
| `second_base` | -0.139 | 0.394 |
| `second_phys` | -0.357 | 0.269 |
| `second_final` | -0.239 | 0.334 |

즉 2등은 ODE-heavy와 같은 방향으로 조금 더 밀어주는 모델이 아니었다. 오히려 많은 샘플에서 다른 방향의 보정을 제공했다.

이 지점이 가장 중요하다.

GOH30 내부 비율을 바꾸는 것은 같은 시스템 안에서 움직이는 것이고, 2등을 섞는 것은 다른 시스템의 판단을 가져오는 것이다.

## 2등 내부 분해로 본 역할

2등을 `base`와 `phys`로 나누어 보면 역할이 더 명확해진다.

| 구성요소 | 회전각 변화 | z 이동 변화 | ODE-heavy와 평균 cos | 해석 |
| --- | ---: | ---: | ---: | --- |
| `second_base` | -0.949도 | -0.000060 | -0.139 | 덜 꺾고 z를 안정화하는 보수적 expert |
| `second_phys` | +0.240도 | -0.000033 | -0.357 | ODE-heavy와 가장 다른 방향의 물리 expert |
| `second_final` | -0.573도 | -0.000070 | -0.239 | base와 phys를 섞은 안정형 외부 expert |

여기서 얻은 결론은 다음이다.

- 덜 꺾는 성향은 주로 `second_base`에서 온다.
- ODE-heavy와 다른 방향의 다양성은 `second_phys`에서 강하게 온다.
- `second_final`은 base의 안정성과 phys의 다양성을 60:40으로 섞은 형태다.

따라서 “2등 코드를 expert로 붙였다”는 말은 더 정확히는 다음과 같다.

```text
GOH30 expert들 위에
2등 base expert와 2등 phys expert가 합쳐진 외부 앙상블 expert를 추가했다.
```

## 왜 두 앙상블을 더하면 기존에 못 보던 것을 보게 되는가

앙상블이 좋아지려면 단순히 모델 수가 많아지는 것만으로는 부족하다. 중요한 것은 서로 다른 오류를 내는 모델을 섞는 것이다.

이번 경우 GOH30과 2등은 다음 차이가 있었다.

| 관점 | GOH30 | 2등 |
| --- | --- | --- |
| 기본 성향 | 최근 움직임 변화에 민감 | 불안정 구간에서 더 보수적 |
| 주요 기준 | 등속 외삽 residual + GRU/ODE/H | Kalman baseline, conservative blend, 회전물리 |
| 회전 구간 | 더 적극적으로 꺾음 | base는 덜 꺾고 phys는 다른 방향성 제공 |
| z축 | 최근 동역학 변화에 반응 | 더 안정적으로 누르는 경향 |
| ODE-heavy와 관계 | 내부 expert 비율 조정 | 외부 expert라 보정 방향이 다름 |

그래서 GOH30이 놓칠 수 있는 부분은 다음이다.

- 최근 회전이 실제 방향 전환인지 노이즈성 흔들림인지 구분하기 어려운 구간
- z축 변화가 실제 상승/하강인지 측정 흔들림인지 애매한 구간
- ODE-heavy가 동역학을 더 믿으면서 과하게 따라가는 구간
- GOH30 내부 expert들이 서로 평균되면서도 같은 방향으로 틀리는 구간

2등은 이런 구간에서 다음을 제공했다.

- `second_base`: 급격한 꺾임과 z 변화를 더 보수적으로 보는 관점
- `second_phys`: GOH30 ODE-heavy와 다른 방향으로 움직이는 물리적 보정 관점
- `second_final`: 두 관점을 과하지 않게 섞은 안정형 예측

결국 2등 expert는 GOH30이 이미 보고 있는 것을 한 번 더 본 것이 아니라, GOH30의 내부 expert들과 상관이 낮은 다른 판단을 제공했다.

이 때문에 단순히 1등 내부 비율을 바꾸는 것보다, 2등 외부 expert를 20% 섞는 쪽이 더 큰 개선으로 이어진 것으로 보인다.

## 최종 조합이 20%였던 이유

2등 final을 20% 섞은 조합이 가장 좋았던 이유는 다음처럼 해석할 수 있다.

- GOH30 ODE-heavy는 이미 강한 주 예측이다.
- 2등 final은 다른 방향성을 주지만, 100% 믿을 만큼 검증된 것은 아니다.
- 20%는 GOH30의 강한 기본 성능을 유지하면서, 2등의 보수적/다양성 보정을 살짝 주입하는 비율이다.

즉 2등을 크게 섞은 것이 아니라, GOH30이 틀릴 가능성이 있는 구간에서만 평균적으로 조금 당겨주는 정도의 역할을 한 것이다.

실제로 최종 예측은 ODE-heavy에서 평균 약 `0.000335`만 이동했다.

| 비교 | 평균 이동 | p95 이동 | 최대 이동 |
| --- | ---: | ---: | ---: |
| `second_final` vs `ODE-heavy` | 0.001675 | 0.004585 | 0.017793 |
| `final` vs `ODE-heavy` | 0.000335 | 0.000917 | 0.003559 |

즉 최종 개선은 큰 좌표 이동으로 만든 것이 아니라, 다른 expert의 방향성을 아주 작게 섞어서 만든 개선이다.

## 최종 해석

이번 방향이 만들어진 이유는 단순히 “2등 코드 결과를 섞으니 점수가 올랐다”가 아니다.

실험적으로는 GOH30 내부 개선이 제한적이었고, 분석적으로는 2등 코드가 GOH30과 다른 행동을 한다는 것이 확인됐다.

특히:

- GOH30은 최근 동역학 변화에 민감하다.
- 2등 base는 불안정한 구간에서 덜 꺾고 z를 안정화한다.
- 2등 phys는 ODE-heavy와 가장 다른 방향의 물리 보정을 제공한다.
- 2등 final은 이 둘을 섞은 안정적인 외부 expert다.
- 이 외부 expert를 20%만 섞으면 GOH30의 장점을 유지하면서, 기존 GOH30 expert들이 못 보던 보수적/다양성 신호를 추가할 수 있다.

그래서 최종 조합은 다음 의미를 갖는다.

```text
GOH30 ODE-heavy:
  강한 기본 예측 + 동역학 강화

2등 final:
  보수적 smoothing + 다른 물리 가정 + 낮은 상관의 외부 expert

최종 앙상블:
  GOH30이 강한 대부분의 샘플은 유지하고,
  GOH30이 과하게 반응할 수 있는 일부 구간만 2등 expert가 보정
```

이것이 1등 앙상블에 2등 앙상블을 더한 방향으로 가게 된 핵심 이유다.
