# 1등/2등 예측 차이 분석 보기 가이드

작성일: 2026-07-01

## 목적

이 문서는 1등 GOH30 원본, ODE 강화 예측, 2등 코드 복원 예측, 최종 블렌딩 예측을 비교해서 어디를 보면 되는지 정리한 것이다.

이번 분석의 핵심 질문은 다음이다.

- 1등 원본과 2등 예측은 어떤 궤적에서 많이 달라지는가?
- ODE 강화는 1등 원본에서 얼마나 움직였는가?
- 최종 `ODE 강화 80% + 2등 20%`는 예측점을 어떤 방향으로 얼마나 이동시키는가?
- 차이가 큰 샘플들이 급회전, 고가속, 고노이즈 같은 구간에 몰리는가?

분석 결과 폴더:

- `outputs/top_expert_comparison/`

## 현재까지의 1차 결론

현재 결과만 놓고 보면 다음 정도는 말할 수 있다.

- 1등 원본과 2등 예측은 대부분의 샘플에서는 가깝다.
- 하지만 회전, 가속, 노이즈가 큰 구간에서는 두 예측의 차이가 커진다.
- 최종 블렌딩은 ODE 강화 예측을 2등 방향으로 약하게 이동시키는 방식이다.
- 따라서 2등 코드의 유용한 힌트는 전체 구간보다는 `회전/가속/노이즈가 큰 궤적을 어떻게 보정하는지`에 있을 가능성이 크다.

### 4. ODE 강화와 2등은 같은 방향으로 움직이는가?

그림:

![ODE 강화와 2등 방향 일치](../../outputs/top_expert_comparison/behavior_ode_second_alignment_by_regime.png)

해석 기준:

- 기준 벡터는 `1등 원본 -> ODE 강화`, `1등 원본 -> 2등`이다.
- 평균 코사인이 양수면 두 이동 방향이 대체로 비슷하다.
- 평균 코사인이 음수면 서로 반대 방향 성향이 더 강하다.

주요 수치:

| 구간 | 같은 방향 비율 | 강한 같은 방향 비율 | 평균 코사인 |
| --- | ---: | ---: | ---: |
| 전체 | 33.4% | 15.7% | -0.239 |
| 고속 | 37.3% | 17.5% | -0.170 |
| 급회전 | 31.6% | 17.0% | -0.273 |
| 최근 회전 | 38.3% | 21.1% | -0.179 |
| 고가속 | 38.2% | 20.6% | -0.169 |
| 고노이즈 | 33.9% | 18.0% | -0.230 |

현재 해석:

- ODE 강화와 2등은 평균적으로 같은 방향으로 움직인다고 보기 어렵다.
- 즉 ODE 강화와 2등 블렌딩은 같은 보정을 두 번 한 것이 아니라, 서로 다른 방향의 약한 보정을 섞은 형태에 가깝다.
- 이것은 2등을 외부 전문가로 섞었을 때 효과가 난 이유와도 맞다. 단순히 ODE 강화의 연장이 아니라 다른 오차 패턴을 보탠 것이다.

## 추가 진단의 1차 결론

현재 test 예측만 놓고 보면 다음 정도는 말할 수 있다.

- 2등은 급회전/고노이즈/낮은 직선성 구간에서 1등보다 덜 꺾는 경향이 있다.
- 2등은 최근 회전/고가속 구간에서는 1등보다 예측 거리를 조금 더 길게 잡는 경향이 있다.
- 2등은 z축 이동을 평균적으로 약간 더 작게 잡고, 특히 수직 변화 구간에서 그 경향이 크다.
- ODE 강화와 2등은 대체로 같은 방향 보정이 아니다. 그래서 2등은 GOH30 내부 비율 조정과 다른 외부 전문가 역할을 한다고 볼 수 있다.

단, 이 분석은 test 정답 없이 예측 간 차이만 본 것이다. 따라서 지금 말할 수 있는 것은 "2등이 이런 식으로 다르게 예측한다"까지이고, "그래서 이쪽이 더 맞다"는 정답 있는 외부폴드 검증에서 봐야 한다.
